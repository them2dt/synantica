# Synantica

Synantica is a curated directory for STEM and research opportunities. The app focuses on clear, data-centric browsing with a calm, editorial layout.

## Getting Started

1. Install dependencies:
   
   ```bash
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

## Project Structure

- `app/` Next.js app routes and layouts
- `components/` UI and feature components
- `lib/` shared utilities, fonts, and hooks
- `styles/` global styles

## Notes

- UI uses a Helvetica-led system with Instrument Serif for headers.
- Design system enforces square edges and thin borders.
